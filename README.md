# Mathematical Logic

This is an introductory mathematical logic textbook written by Richard Grandy, Michael Barkasi, and Joshua Reagan. It is freely distributed under a Creative Commons license. Please feel free to download and use MathLogic.pdf for self-study or classroom use. A version of this text has been used for logic instruction at Rice University since 2012. [Click here to download as a pdf.](https://raw.githubusercontent.com/joshuareagan/logic_text/main/MathLogic.pdf)
